🏗 VRF H/R System — 3 Line Types

  • Liquid line — ท่อสารทำความเย็นสถานะ liquid เชื่อมต่อ Outdoor → RB → Indoor. แรงดันสูง (HP side) ขณะ cooling. ต้องใช้ mesh ละเอียดที่สุด เพราะ EEV/TXV ทนสิ่งสกปรกได้น้อย
  • Suction line — ท่อ gas แรงดันต่ำ (LP side) จาก Indoor → RB → Outdoor compressor. ขณะ cooling mode. Density ต่ำมาก velocity สูง ใช้ mesh กลาง
  • Discharge line — ท่อ gas แรงดันสูงอุณหภูมิสูง จาก Outdoor compressor → RB (H/R system). ขณะ heating mode หรือ simultaneous. ต้องใช้ strainer ทนแรงดันสูง

⚙ Strainer Components

  • Body (K_body) — ค่า loss coefficient ของตัวถัง strainer ค่าทั่วไป K=3–8 สำหรับ Y-strainer clean
  • Screen element — ตาข่ายกรองแบบ cylindrical ทำจาก SS 304/316. ΔP ขึ้นกับ mesh, ความยาว, และ flow rate
  • FAR (Free Area Ratio) — สัดส่วนพื้นที่เปิดต่อพื้นที่ทั้งหมด ยิ่ง mesh ละเอียด FAR ยิ่งต่ำ ΔP ยิ่งสูง
  • V_screen — ความเร็วผ่านรูตาข่าย ต้องไม่เกินค่า limit เพื่อป้องกัน ΔP สูงเกินและ mesh deformation

📏 Screen Length Rule of Thumb

  • L_min = Q_v ÷ (V_scr_max × π × D_scr × FAR)
  • Practical minimum: L ≥ 1.5 × pipe_ID
  • ยาวขึ้น = ΔP น้อยลง + dirt capacity มากขึ้น
  • Typical: L = 2–4 × pipe_ID
  • ท่อ ¼"–½": ใช้ in-line filter, screen L ≈ 30–60 mm
  • ท่อ ⅝"–1": Y-strainer, screen L ≈ 60–120 mm
  • ท่อ 1⅛"+: Y/basket strainer, screen L ≈ 100–200 mm

⚠ Limits & Status Colors

  • Pipe velocity
    Liquid ≤1.5 m/s OK | ≤2.5 warn
    Gas ≤15 m/s OK | ≤20 warn
  • Screen velocity
    Liquid ≤0.25 m/s OK | ≤0.35 warn
    Suction ≤2.5 m/s OK | Gas ≤2.0 m/s
  • ΔP body ≤20 kPa OK | ≤30 warn
  • ΔP total ≤25 kPa OK | ≤40 warn
  • ถ้า Warning → ใช้ท่อ/strainer ใหญ่ขึ้น หรือ mesh coarser
  • ถ้า Over limit → ต้องแก้ไขก่อน final design
